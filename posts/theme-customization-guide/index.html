主题自定义指南

一份完整的 Modern Archivist 主题自定义指南。列出修改不同内容时，应当编辑哪个文件的哪个位置。

本文是主题 Modern Archivist 的自定义指南。当你需要修改站点的某个部分时，可以按照下方索引找到对应的文件。

站点全局配置

所有站点级别的设置都在根目录的 hugo.toml 中。

站点标题与描述

# hugo.toml
title = 'The Modern Archivist'

[params]
  description = '站点描述，用于 SEO 和 meta 标签'

首页标题与副标题

# hugo.toml → [params]
homeTitle = '显示在首页的大标题'
homeSubtitle = '首页标题下方的介绍文字'

导航菜单

增减菜单项，修改 hugo.toml 中的 [[menu.main]] 块：

# hugo.toml
[[menu.main]]
  name = 'Home'       # 显示的文字
  url = '/'            # 链接地址
  weight = 1           # 排序权重，越小越靠左

[[menu.main]]
  name = 'Archives'
  url = '/archives/'
  weight = 2

[[menu.main]]
  name = 'About'
  url = '/about/'
  weight = 3

社交链接

页脚的社交链接由 hugo.toml 中的 [params.social] 控制：

# hugo.toml → [params.social]
github = 'https://github.com/your-username'
scholar = 'https://scholar.google.com/...'
twitter = 'https://twitter.com/...'

代码高亮主题

# hugo.toml → [markup.highlight]
style = 'dracula'    # 可选: monokai, github, solarized-dark, nord 等

完整的可用主题列表参见 Hugo Chroma Styles。

字体

字体配置在 两个位置：

1. 引入字体文件

themes/modern-archivist/layouts/partials/head.html

找到 Google Fonts 的 <link> 标签，替换为你需要的字体：

<link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=你的字体名&display=swap" />

2. 指定字体族

同一文件中的 Tailwind 配置和 <style> 块：

// head.html → tailwind.config → theme.extend.fontFamily
fontFamily: {
  headline: ["Noto Serif", "serif"],   // 标题字体
  body: ["Inter", "sans-serif"],       // 正文字体
  label: ["Inter", "sans-serif"]       // 标签/小字字体
}

/* head.html → <style> */
body { font-family: 'Inter', sans-serif; }
h1, h2, h3, h4, h5, h6 { font-family: 'Noto Serif', serif; }

配色方案

所有颜色 token 定义在：

themes/modern-archivist/layouts/partials/head.html

找到 tailwind.config 中的 colors 对象。主要颜色含义：

Token 用途 当前值
primary 主题色（链接、按钮、标签） #3e6471
on-background 正文主色 #2d3339
on-surface-variant 次要文字（正文、标签） #596066
background 页面背景 #f9f9fc
surface-container-low 卡片/区块背景 #f2f3f7
primary-container 选中文本背景 #c2e9f9

要修改主题色风格，只需替换 primary 及其相关 token 即可。

正文排版样式

标题层级、段落间距、代码块、引用框、表格等所有正文排版样式在：

themes/modern-archivist/layouts/partials/head.html → <style> 块

关键的 CSS 选择器对照表：

要修改的元素 CSS 选择器
二级标题 (H2) .prose h2
三级标题 (H3) .prose h3
四级标题 (H4) .prose h4
正文段落 .prose p
引用块 .prose blockquote
行内代码 .prose code:not(pre code)
代码块 .prose pre
表格头 .prose thead th
表格单元格 .prose tbody td
链接 .prose a
暗色模式覆盖 html.dark .prose h2 等

页面布局模板

导航栏

themes/modern-archivist/layouts/partials/header.html

包括：固定顶栏、Logo、桌面菜单、暗色模式按钮、移动端汉堡菜单。

页脚

themes/modern-archivist/layouts/partials/footer.html

包括：站点名称、版权文字、社交链接。

HTML <head>

themes/modern-archivist/layouts/partials/head.html

包括：CDN 引入、Tailwind 配置、颜色 token、字体、KaTeX、所有 CSS 样式。

基础包装模板

themes/modern-archivist/layouts/_default/baseof.html

所有页面共用的 HTML 骨架：<html> → <head> → <body> → header + main + footer + JS。

各页面对应文件

首页

层 文件
内容 content/_index.md
模板 themes/modern-archivist/layouts/index.html

文章详情页

层 文件
内容 content/posts/你的文章.md
模板 themes/modern-archivist/layouts/_default/single.html

模板中包含：侧边栏 TOC、文章头部（日期/阅读时间/标签）、正文 .prose 容器、上下篇导航。

About 页

层 文件
内容 & 数据 content/about.md（front matter 中的结构化数据 + Markdown 正文）
模板 themes/modern-archivist/layouts/_default/about.html

About 页的 front matter 参数：

name: "你的名字"
role: "你的头衔"
avatar: "/images/avatar.jpg"     # 图片放在 static/images/
email: "you@example.com"
social:
  twitter: "https://..."
  github: "https://..."
  linkedin: "https://..."
research_interests:
  - "兴趣 1"
  - "兴趣 2"
education:
  - degree: "学位名称"
    school: "学校, 年份"
publications:
  - year: "2024"
    type: "JOURNAL"
    title: "论文标题"
    description: "描述"
    tags: ["Peer Reviewed"]
cv_url: "https://..."
inquiries_text: "联系说明文字"

归档页

层 文件
内容 content/archives.md
模板 themes/modern-archivist/layouts/_default/archives.html

文章列表页（如标签页、分类页）

层 文件
模板 themes/modern-archivist/layouts/_default/list.html

暗色模式

暗色模式逻辑分布在两个文件中：

功能 文件
初始化（防闪烁） head.html 中的内联 <script>
切换 & 持久化 themes/modern-archivist/static/js/main.js
暗色模式 CSS head.html 中 html.dark .prose ... 等规则
Tailwind 暗色类 各模板中的 dark:bg-slate-900 等 class

静态资源

资源 存放位置
头像图片 static/images/avatar.jpg
自定义 JS themes/modern-archivist/static/js/main.js
其他图片 static/images/ 目录下，文章中用 /images/xxx.png 引用
favicon static/favicon.ico（需自行添加）

数学公式

数学渲染由 KaTeX 提供，CDN 引用在 head.html 中。支持的语法：

  • 行内公式：$E = mc^2$ → $E = mc^2$
  • 块级公式：$$..$$ 独占一行

Hugo 已配置 passthrough 扩展（hugo.toml），确保 $ 和 $$ 内的内容不被 Markdown 解析器处理。
